# Run the Service

You can run the application using ./gradlew bootRun.

Or you can build the JAR file using ./gradlew build. Then you can run the JAR file: java -jar build/libs/gs-rest-service-0.1.0.jar

# Test the service

Visit http://localhost:8090/greeting or http://localhost:8090/greeting?name=Marcin
